# Notas y pendientes para más adelante

## Mi diario

### Qué es (ahora)
- **Mi diario** = blog público de **microtextos motivadores diarios**.
- Entradas cortas, una (o unas pocas) al día, tipo reflexión / frase / pequeño artículo motivador.
- **No requiere** usuario logueado ni backend. Contenido estático o gestionado por CMS/archivos.

### Pendiente para más adelante
Cuando exista **soporte para ligar la info del usuario a la web** (auth, backend, base de datos):

- **Diario personal / dashboard del usuario:** rachas, respuestas a la pregunta diaria, progreso en carreras, historial, hitos, etc.
- Posible convivencia: **blog público** "Mi diario" (motivación diaria) + **área privada** "Mi cuenta" / "Mi progreso" con datos del usuario.
